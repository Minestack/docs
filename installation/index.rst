Installation
============

The Minestack system consists of two key services; the Identity Service and the Compute Service. These services
use other supporting services that include Docker, RabbitMQ, Consul, Vault and a SQL based Relational Database.

This guide will show you how to install Minestack on a CentOS 7 based Linux distribution. Sample configuration files
and explanations of configuration options are included.
